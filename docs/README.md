# Documentation

This directory contains project documentation for GEPA Mindfulness Superalignment,
including alignment foundations, training notes, execution guides, and research
background for the repository.

## GEPA Mindfulness Constitution

The canonical constitution is
[docs/GEPA_Mindfulness_Constitution.md](GEPA_Mindfulness_Constitution.md).
It is the repository's virtue-oriented alignment artifact: a source text for
principled reasoning, value negotiation, deep alignment, and metacognitive
training grounded in human prosperity, reduced suffering, scientific knowledge,
and autonomy.

The constitution is a living document. It should be refined as the project
learns from experiments, evaluations, deployment experience, and stakeholder
feedback, while preserving the existing file as the canonical source of truth.
It is not a rigid rulebook, system prompt, legal policy, safety policy, or
standalone deployment guardrail.

When stricter system instructions, safety policies, legal requirements, or
deployment rules apply, those stricter rules override the constitution. The
constitution's role is to help models reason with values and virtues, not merely
imitate approved outputs or memorize surface behaviors.
